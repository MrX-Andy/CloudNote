<h1>CloudNote</h1>  
云签 二维码扫描应用


<h2>Description: </h2>

	基于Google Zxing进行开发，数据保存在云端，用户扫描二维码可以编辑，读取便签，实现发送及接受来自他们真诚的祝福。

<h2>Tips: </h2>   

	此项目部分idea以及source code来源自互联网，目的仅仅是为了学习，不做任何其他的目的。
